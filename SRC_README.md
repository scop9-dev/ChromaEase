# ChromaEase Source

Welcome to the source folder of **ChromaEase**!  

Feel free to browse the code and see how the tweening system is implemented.  
This folder contains the core classes and utilities, including:

- `Tweener` – main entry point for numeric and color tweens  
- `Easing` – built-in easing functions  
- Internal runner and tween classes  

If you have any questions, issues, or suggestions, please check the [NuGet Issues discussions](https://github.com/scop9-dev/ChromaEase/discussions) 

Happy coding!
